# STEP-09: Боковая панель записи (детали + действия)

## Зависимости

- **STEP-06**: Сетка расписания (клик на блок записи).
- **STEP-07**: Данные записи (booking object с сегментами).

## Scope

При клике на блок записи в сетке — открывается боковая панель справа с деталями и действиями.

## UI

```
┌─── Детали записи ────────────────┐
│                              [✕] │
│  Иванова Мария                   │
│  +7 (999) 123-45-67              │
│                                  │
│  ─────────────────────────────── │
│  Процедура: Тайский массаж 1ч   │
│  Время: 10:30 — 11:30            │
│  Кабинка: Кабинка 1              │
│  Мастеров: 1 │ Клиентов: 1       │
│  ─────────────────────────────── │
│                                  │
│  Для комбо — timeline всех шагов:│
│  ● Сауна       10:30–11:30      │
│  ● Тайский 1ч  11:30–12:30      │
│  ● Ойл 1ч      12:30–13:30      │
│                                  │
│  ─────────────────────────────── │
│  Итого: 5 000 ₸                  │
│                                  │
│  Статус: [● Забронировано ▾]     │
│                                  │
│  Заметки: Аллергия на масло      │
│                                  │
│  [🗑 Удалить запись]              │
└──────────────────────────────────┘
```

### Характеристики:

- **Ширина:** 360px, фиксирована справа.
- **Анимация:** slide-in справа (transform: translateX(100%→0), 250ms ease-out).
- **Фон:** #1A2332.
- **Left-border:** 2px solid #D4A84B.
- **Закрытие:** кнопка ✕ или клик вне панели.

### Данные:

- Имя клиента (крупно, 16px, вес 600).
- Телефон (14px, #8A9AAE, кликабельный — tel: link).
- Разделитель.
- Для single_procedure: название, время, кабинка, мастеров, клиентов.
- Для combo: timeline шагов (вертикальный список с цветными точками по категории). Каждый шаг: ● цвет + название + время.
- Разделитель.
- Цена (крупно, 18px, вес 700, цвет #D4A84B).

### Статус (dropdown):

- Текущий статус показан цветным бейджем.
- Клик → dropdown: booked (золотой), completed (зелёный), cancelled (красный), no-show (жёлтый).
- При смене — сразу сохраняется в storage. Toast "Статус изменён".

### Удаление:

- Кнопка "Удалить запись" — красная, внизу панели.
- Клик → подтверждение: "Удалить запись Ивановой Марии на 10:30?" с кнопками "Да, удалить" / "Отмена".
- При удалении — убрать из storage, закрыть панель, обновить сетку. Toast "Запись удалена".
