# STEP-10: Журнал клиентов

## Зависимости

- **STEP-01**: Storage layer, bookings data.
- **STEP-07**: Записи существуют в storage (журнал строится из bookings).

## Scope

Таб "Журнал" — таблица всех записей с фильтрами, поиском, сортировкой, inline-сменой статуса.

## Данные

Журнал НЕ хранится отдельно. Строится из bookings:
- Загрузить все ключи `spa-crm:bookings:*` (оба салона, все месяцы).
- Для этого: `Storage.list('spa-crm:bookings:')` → получить все ключи → загрузить каждый.
- Объединить в плоский массив, отсортировать по дате (новые сверху).

### Столбцы таблицы:

| Столбец      | Данные                         | Сортировка |
|-------------|-------------------------------|-----------|
| Дата        | DD.MM.YYYY                    | ✓         |
| Имя         | clientName                    | ✓         |
| Телефон     | clientPhone                   | —         |
| Салон       | salon.name                    | ✓         |
| Услуга      | procedure/combo name          | ✓         |
| Длительность| totalDuration мин              | ✓         |
| Цена        | totalPrice ₸                  | ✓         |
| Статус      | бейдж (кликабельный dropdown) | ✓         |

### Фильтры (строка над таблицей):

- **Поиск** — text input с иконкой 🔍. Фильтрует по clientName и clientPhone (contains, case-insensitive).
- **Салон** — dropdown: Все / Салон 1 / Салон 2.
- **Статус** — dropdown: Все / Забронировано / Выполнено / Отменено / Не пришёл.
- **Даты** — два date inputs: от и до.

### Сортировка:

- Клик по заголовку столбца → сортировка asc/desc.
- Визуальный индикатор: стрелка ↑↓ рядом с активным столбцом.
- По умолчанию: дата descending.

### Inline-смена статуса:

- Бейдж статуса кликабельный → dropdown (booked/completed/cancelled/no-show).
- Цвета бейджей: booked #D4A84B, completed #4ADE80, cancelled #F87171, no-show #FBBF24.
- При смене → сохранить в storage. Toast.

### Пагинация:

- 25 записей на страницу.
- Внизу: "Показано 25 из 142" + кнопки [< 1 2 3 ... >].

## UI-стиль таблицы:

- Header: фон #1A2332, текст uppercase 11px #8A9AAE, letter-spacing 0.5px.
- Строки: чередование #1A2332 / #151E2B.
- Hover: #1D2A3A.
- Фильтры: inputs стиль из палитры (фон #0F1419, border #2A3A4E).
