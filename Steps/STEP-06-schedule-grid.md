# STEP-06: Расписание — Дневная сетка

## Зависимости

- **STEP-01**: Storage layer, типы данных (Booking, Salon, Room), bookings state.
- **STEP-03**: Настройки салона (rooms, workStart, workEnd, dayOff, hasSauna).
- **STEP-04**: Процедуры (для цветов блоков по категории).
- **STEP-05**: Комбо (для отображения комбо-сегментов).

## Scope

Таб "Расписание" — дневная сетка с отображением существующих записей. Без формы создания записи (это STEP-07).

## Что на экране

### Верхняя панель:

```
[< Пред]  Понедельник, 16 марта 2026  [След >] [📅]     [+ Новая запись]
```

- Навигация по дням: кнопки Пред/След.
- Иконка календаря: открывает date picker (простой — input type="date" стилизованный или custom).
- Если выбран выходной: вместо сетки — "Выходной день (Понедельник)" с иконкой Moon.
- Кнопка "+ Новая запись" — заглушка (функциональность в STEP-07).

### Сетка:

```
┌────────┬───────┬───────┬───────┬───────┬───────┬─────────┐
│        │ 10:00 │ 10:30 │ 11:00 │ 11:30 │ 12:00 │  ...    │
├────────┼───────┼───────┼───────┼───────┼───────┼─────────┤
│Кабинка1│       │ ██████████████████████│       │         │
├────────┼───────┼───────┼───────┼───────┼───────┼─────────┤
│Кабинка2│       │       │       │       │       │         │
├────────┼───────┼───────┼───────┼───────┼───────┼─────────┤
│Сауна   │       │       │       │       │       │         │
├────────┴───────┴───────┴───────┴───────┴───────┴─────────┤
│  Мастера: ■■■■□□  ■■■■■□  ■■□□□□  ...                    │
└──────────────────────────────────────────────────────────┘
```

**Ось Y (левый столбец, 120px, фиксирован):**
- Кабинки салона (из salon.rooms).
- Сауна (если hasSauna = true).

**Ось X (горизонтальный скролл):**
- Время от workStart до workEnd, шаг 30 мин.
- Каждый столбец: 80px ширина.

**Ячейки:**
- Высота: 60px.
- Фон: #151E2B.
- Hover на пустой: #1D2A3A + cursor pointer.
- Клик на пустой слот: пока ничего (STEP-07 добавит предзаполнение формы).

**Блоки записей:**
- Загрузить записи текущего дня: `spa-crm:bookings:{salonId}:{YYYY-MM}`, фильтр по date.
- Каждый сегмент записи — блок, позиционированный абсолютно:
  - left: (startTime - workStart) / 30min × 80px.
  - width: duration / 30min × 80px.
  - Цвет по категории: massage #2D6A4F, sauna #B85C38, peeling #7B68AE.
  - border-radius 6px.
  - Внутри: имя клиента (truncate), название процедуры мелко (11px).
- Комбо: сегменты разных цветов + обёртка с золотой рамкой (1px dashed #D4A84B33).
- Пиллинг внутри сауны: бейдж "+ Пиллинг" (лавандовый) в правом верхнем углу блока сауны.
- Hover: brightness(1.1) + box-shadow, transition 150ms.
- Клик на блок: пока ничего (STEP-09 добавит боковую панель).

**Текущее время:**
- Если выбран сегодняшний день: вертикальная линия 2px красная, полупрозрачная, на позиции текущего времени.

**Индикатор мастеров (под сеткой):**
- Для каждого 30-минутного слота: посчитать суммарный therapistCount из всех сегментов, пересекающихся с этим слотом.
- Отобразить: маленькие квадраты (■ занят = #D4A84B, □ свободен = #2A3A4E).
- Текст: "4/6" (занято/всего).
- Если все заняты — число красным (#F87171).

## Расчёт занятости мастеров в слоте

```javascript
function getTherapistUsage(bookings, slotStart, slotEnd, salon) {
  let used = 0;
  for (const booking of bookings) {
    for (const seg of booking.segments) {
      if (seg.startTime < slotEnd && seg.endTime > slotStart) {
        used += seg.therapistCount;
      }
    }
  }
  return { used, total: salon.therapistCount };
}
```

## Acceptance Criteria

- [ ] Сетка рендерится с кабинками и сауной по оси Y.
- [ ] Временная шкала от workStart до workEnd, шаг 30 мин.
- [ ] Навигация по дням (пред/след/datepicker).
- [ ] Выходной день — показывается заглушка.
- [ ] Существующие записи отображаются как цветные блоки правильной ширины и позиции.
- [ ] Цвета блоков по категории.
- [ ] Комбо — связанные сегменты с золотой рамкой.
- [ ] Бейдж пиллинга на блоке сауны.
- [ ] Индикатор мастеров под сеткой.
- [ ] Вертикальная линия текущего времени (только сегодня).
- [ ] Горизонтальный скролл работает, левый столбец фиксирован.
