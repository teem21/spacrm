# STEP-02: Онбординг — Визард первого запуска

## Зависимости

- **STEP-01**: Storage layer, DEFAULT_SALONS, DEFAULT_PROCEDURES, app shell, `needsOnboarding` state.

## Scope

Пошаговый визард при первом запуске (когда `spa-crm:salons` отсутствует в storage). Собирает конфигурацию обоих салонов и сохраняет в storage.

## Шаги визарда (5 шагов)

**Шаг 1:** Название салона 1.
**Шаг 2:** Кабинки салона 1 — сколько кабинок (1-6), для каждой — кол-во кроватей (1 или 2).
**Шаг 3:** Мастера + часы салона 1 — кол-во мастеров (1-15), рабочие часы (workStart/workEnd), выходной день, буфер (5-60 мин).
**Шаг 4:** Сауна/пиллинг салона 1 — есть ли сауна (toggle), если да: вместимость, длительность. Есть ли пиллинг: макс мастеров, время на человека, макс людей.
**Шаг 5:** Повторить шаги 1-4 для салона 2 (в упрощённом виде — все поля на одном экране).

### При завершении:

1. Сохранить `spa-crm:salons` с конфигурацией обоих салонов.
2. Создать стандартные процедуры (11 шт.) для каждого салона → `spa-crm:procedures:salon-1` и `salon-2`.
3. Установить `needsOnboarding = false`, загрузить данные, перейти на главный экран.

```
│  ◈ Добро пожаловать в SPA CRM       │

│  Как он называется?                  │
│  [ Thai Orchid              ]        │
│  ← Назад              [ Далее → ]   │
└──────────────────────────────────────┘
```

- Полноэкранный, центрированный (max-width 480px).
- Фон: #0F1419. Карточка: #1A2332, border-radius 12px, padding 32px.
- Прогресс-бар: 5 сегментов. Заполненные — #D4A84B, незаполненные — #2A3A4E.
- Кнопка "Далее" — Primary (золотая). "Назад" — текстовая ссылка #8A9AAE.
- Inputs: стиль из палитры (фон #0F1419, border #2A3A4E, focus #D4A84B).
- Toggle для сауны/пиллинга: switch-стиль.

## Acceptance Criteria

- [ ] При первом запуске (нет данных в storage) показывается визард.
- [ ] 5 шагов с навигацией вперёд/назад.
- [ ] Прогресс-бар отражает текущий шаг.
- [ ] Валидация: название не пустое, кол-во кабинок 1-6, мастеров 1-15.
- [ ] При завершении — данные салонов сохранены в storage.
- [ ] Стандартные процедуры (11 шт.) созданы для каждого салона.
- [ ] После онбординга — переход на главный экран (Расписание).
- [ ] Повторный запуск НЕ показывает визард (данные уже есть).
